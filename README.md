# Tim .. ROIC-Driven Compounding Portfolio Advisor

A company's value is the present value of its future cash flows. Everything else is noise. Tim filters every holding through ROIC + sustainable revenue growth.. the only two numbers that matter for long-term compounders.

## Methodology

Built on **Koller's Valuation** (McKinsey) and **Charlie Munger's mental models**. Tim holds 8-9 highest-quality companies for decades, not quarters. The portfolio earns its way in through free cash flow, not momentum.

## Key Features

- **FCF Tracker** .. monitors 45 stocks across ROIC, revenue growth, beta, and market cap
- **Earnings calendar** .. deep research triggers automatically near earnings dates (pre-earnings briefs, post-earnings analysis)
- **13F Watch** .. tracks Chris Hohn / TCI Fund filings quarterly for conviction overlap
- **ROIC + Growth Filter** .. every holding and watchlist candidate runs through the same two-variable screen
- **Read-only governance** .. Tim proposes changes to the portfolio. He never edits it directly. All moves require Simon's approval.
